🤔 O que é Álgebra?

A Álgebra é carinhosamente chamada de Matemática das Letras. Mas não precisa ter medo! As letras na matemática funcionam como "caixas vazias" esperando que um número seja colocado dentro delas.

👕 Por que usamos letras?

Imagine uma fábrica de camisetas. O lucro do dono é de R$ 20,00 por cada camiseta vendida. Como não sabemos o futuro e não fazemos ideia de quantas serão vendidas hoje, usamos a letra x para representar essa quantidade desconhecida.

Assim, criamos uma fórmula geral: Lucro = $20x$

🗣️ Tradução para o "Matematiquês"

Aprender Álgebra é como aprender um novo idioma. Veja como traduzimos frases do português para a matemática de forma simples:

  • O dobro de um número $\rightarrow$ $2x$
  • A metade de um número $\rightarrow$ $x/2$
  • Um número aumentado de 3 $\rightarrow$ $x + 3$
💡 Macete: A Regra do Olhou-Escreveu

Sempre que o problema disser as palavras "UM NÚMERO", esse é o seu gatilho mental. Não pense duas vezes: escreva imediatamente a letra X no papel. Esse é o primeiro passo para destravar qualquer questão!
